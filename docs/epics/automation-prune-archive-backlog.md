# automation-prune-archive-backlog (epic)
**Description:** Automate pruning and archiving of completed or obsolete backlog items. The system should automatically move completed/obsolete items to .nootropic-cache/backlog-archive.json, keeping the main backlog actionable and up to date. Reference scripts/pruneBacklogTodos.ts, scripts/backlogSync.ts, and scripts/generateBacklogSummary.ts.
